# iOS - Ojo Claro AI

Base iOS nativa con SwiftUI.

## Opción recomendada

Instalar XcodeGen y generar proyecto:

```bash
brew install xcodegen
cd ios
xcodegen generate
open OjoClaroIOS.xcodeproj
```

## Permisos incluidos

- Cámara.
- Micrófono.
- Reconocimiento de voz.
- Ubicación en uso.

## Primer objetivo

Abrir HomeView, reproducir voz, llamar al backend local y luego conectar cámara real.
