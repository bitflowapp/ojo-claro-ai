# GitHub Pages para Ojo Claro AI

## Pasos concretos

1. Subí la rama `main` al repositorio de GitHub.
2. Abrí el repositorio en GitHub.
3. Entrá en **Settings**.
4. Abrí **Pages**.
5. En **Build and deployment** elegí:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/docs`
6. Guardá los cambios.
7. Esperá unos minutos a que GitHub publique el sitio.

## URL esperada

La URL final suele ser parecida a:

`https://marcoluna-nqn.github.io/ojo-claro-ai/`

## Cómo actualizar

- Editá `docs/index.html`, `docs/styles.css` o `docs/app.js`.
- Hacé commit y push a `main`.
- GitHub Pages se actualiza solo.
