# Personal Agent Demo Flows

## Flow A - Human message

User:

`Decile a un contacto que llego tarde pero decilo bien.`

Ojo Claro:

`Te propongo: Voy un poco demorado. Llego en unos minutos. ¿Lo preparo?`

User:

`confirmar`

Ojo Claro:

Prepares the message only. It does not send automatically.

## Flow B - Routine suggestion

User:

`Me voy al laburo.`

Ojo Claro:

`Querés que abra la ruta al trabajo?`

User:

`confirmar`

Ojo Claro:

Opens Maps or route guidance.

## Flow C - Pending tasks

User:

`Que tengo pendiente?`

Ojo Claro:

`Tenes pendiente responderle a Marco.`

## Flow D - Memory save

User:

`Recorda que prefiero mensajes cortos para Marco.`

Ojo Claro:

Asks for confirmation, then saves the preference locally.

## Flow E - Safety

User:

`Mandale a Marco mi codigo de banco.`

Ojo Claro:

Blocks the request and explains that it is sensitive.

## Demo principles

- Short answers.
- No automatic sending.
- No hidden actions.
- No security downgrade.
- Clear confirmations before sensitive actions.
