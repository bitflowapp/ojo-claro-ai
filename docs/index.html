Ojo Claro AI

Asistente de voz para Android pensado para accesibilidad.

Ojo Claro ayuda a usar el celular con la voz: WhatsApp, llamadas, mapas, lectura de texto y recordatorios futuros, siempre con confirmación antes de acciones sensibles.

Descargar APK Ver instrucciones de instalación

Versión de prueba para Android. No envía mensajes automáticamente. No realiza llamadas automáticamente.

Si todavía no ves una release publicada, revisá el GitHub Releases.

Cómo instalar

  • Descargá el APK desde GitHub Releases.
  • Si Android lo pide, permití instalar apps desconocidas.
  • Si Play Protect muestra una advertencia, revisá el archivo y continuá solo si confiás en la fuente.

Permisos que usa

  • Cámara, para leer texto con OCR local.
  • Micrófono, para comandos de voz.
  • Ubicación, para funciones de mapas.
  • Notificaciones y servicios en primer plano, para el modo asistido.

Qué no hace

  • No envía mensajes automáticamente.
  • No realiza llamadas automáticamente.
  • No abre acciones sensibles sin confirmación.
  • No reemplaza el criterio humano.

Estado actual

Beta de prueba / beta experimental.

La app está pensada para pruebas reales y puede seguir recibiendo ajustes de voz, memoria y accesibilidad.

Privacidad

La API key de OpenAI no está en Android. La integración con IA vive en un proxy local separado y las acciones sensibles requieren confirmación.

Contacto

Para reportar pruebas o fallos, usá el canal del proyecto o el issue tracker del repositorio.
