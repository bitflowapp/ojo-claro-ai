# Instalar Ojo Claro AI en Android

1. Abrí la página del proyecto o la release de GitHub.
2. Descargá `ojo-claro-ai-release.apk`.
3. En Android, permití la instalación de apps desconocidas para la app desde la que abrís el archivo. Android puede mostrar un aviso y pedirte confirmar esa autorización.
4. Si Play Protect muestra una advertencia, revisá que el APK venga del repositorio oficial y del tag de release correcto antes de continuar. Si no confiás en el archivo, cancelá la instalación.
5. Instalá el APK.
6. Abrí Ojo Claro y concedé los permisos que la app pida para cámara, micrófono, ubicación y notificaciones, según el flujo que uses.
7. Si instalás una versión nueva firmada con otra clave, desinstalá la versión anterior primero.

## Si algo falla

- Verificá que el archivo descargado termine en `.apk`.
- Probá volver a descargarlo.
- Asegurate de tener espacio suficiente en el teléfono.
- Si la app no abre, enviá el mensaje de error o una captura del logcat del dispositivo de prueba.

## Permisos esperados

- Cámara: para OCR local.
- Micrófono: para comandos de voz.
- Ubicación: para mapas y ubicación actual.
- Notificaciones: para avisos del asistente.

## Qué no hace

- No envía mensajes automáticamente.
- No realiza llamadas automáticamente.
- No usa cuentas bancarias ni claves.
- No se publica todavía en Play Store.
